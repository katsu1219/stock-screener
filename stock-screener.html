📖 Gemini APIキーの取得方法（無料・5分）

1. Google AI Studio にアクセス（Googleアカウントでログイン）
2. 「Get API key」→「Create API key」をクリック
3. 表示されたキー（AIzaSy...で始まる文字列）をコピー
4. 上の入力欄に貼り付けて「保存する」を押す

⚠️ 無料枠：Gemini 2.0 Flashは1分15リクエスト、1日1,500リクエストまで無料。株の分析には十分です。

🔒 セキュリティ：APIキーはこのブラウザのローカルストレージにのみ保存されます。外部サーバーには送信しません。

🔄 AI自動切り替え設定

チェックを入れると、Claudeが利用制限に達した場合などにGeminiへ自動で切り替わります。
切り替えが起きた際は分析結果の上部に「Gemini使用」と表示されます。
